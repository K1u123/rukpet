# Система бронирования путешествий

Эта система позволяет пользователям искать рейсы, бронировать билеты и управлять маршрутами.  Приложение включает в себя административную панель для управления данными о рейсах, а также удалять и добавлять новые билеты.

## Функциональность:

Поиск рейсов: Пользователи могут искать рейсы по заданным критериям (пункт отправления, пункт назначения, дата).
Бронирование билетов: После поиска, пользователи могут бронировать найденные билеты.
Управление маршрутами: Администраторы могут добавлять, удалять и изменять маршруты.


## Диаграммы BPMN:

В проекте представлены диаграммы BPMN, описывающие процессы:

'Процесс бронирования билета.bpmn': Описывает процесс бронирования билета пользователем.
'Процесс поиска и вывода списка билетов (1).bpmn':  Описывает процесс поиска и вывода списка доступных билетов.
'Процесс управления билетами и маршрутами.bpmn':  Описывает процесс управления билетами и маршрутами администратором.


## Техническая реализация:

Система реализована на языке Python.  Код использует объектно-ориентированный подход.  Основные классы:

'Ticket': Представляет собой билет на рейс.  Хранит информацию о номере билета, пункте отправления, пункте назначения, дате и времени.
'Admin': Класс, предоставляющий функциональность для администратора, например, добавление, удаление и изменение билетов.
'LoggedAdmin': Класс, обрабатывающий вход администратора с проверкой пароля.
'BookingSystem': Класс, реализующий функциональность системы бронирования для пользователей: поиск билетов, бронирование и отображение списка билетов.
